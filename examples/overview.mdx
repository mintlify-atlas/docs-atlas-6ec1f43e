The fastest way to learn Ebitengine is to run real code. Every example in this section is a self-contained Go program that compiles and runs with a single command — no project setup required. Browse by category below, or pick any example name and run it directly.

## Running an example

Any Ebitengine example can be run without cloning the repository. Replace `EXAMPLE_NAME` with the name of the example you want to run:

```bash
go run github.com/hajimehoshi/ebiten/v2/examples/EXAMPLE_NAME@latest
```

For instance, to run the 2048 sliding tile puzzle:

```bash
go run github.com/hajimehoshi/ebiten/v2/examples/2048@latest
```

<Tip>
  You need Go 1.21 or later installed. On Linux, install the required system packages for your display server (X11 or Wayland) if you haven't already.
</Tip>
