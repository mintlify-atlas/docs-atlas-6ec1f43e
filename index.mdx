Ebitengine (formerly Ebiten) is an open source, dead simple 2D game engine for the Go programming language. With a straightforward API built around a single `Game` interface, you can build games that run on desktop, mobile, and the web without changing your code.

## Why Ebitengine?

Ebitengine gives you a minimal, productive surface area for game development in Go. You implement one interface, call `ebiten.RunGame`, and Ebitengine handles the rest — windowing, rendering, input, and audio — across every supported platform.

## Get started in 3 steps

<Steps>
  <Step title="Install Go and Ebitengine">
    Make sure you have Go 1.21 or later, then add Ebitengine to your module:

    ```bash
    go get github.com/hajimehoshi/ebiten/v2
    ```
  </Step>
  <Step title="Implement the Game interface">
    Create a struct that satisfies `ebiten.Game` with `Update`, `Draw`, and `Layout` methods:

    ```go main.go
    type Game struct{}

    func (g *Game) Update() error { return nil }
    func (g *Game) Draw(screen *ebiten.Image) {
        screen.Fill(color.RGBA{0x1a, 0x1a, 0x2e, 0xff})
    }
    func (g *Game) Layout(outsideWidth, outsideHeight int) (int, int) {
        return 640, 480
    }
    ```
  </Step>
  <Step title="Run your game">
    Pass your game to `ebiten.RunGame`:

    ```go main.go
    func main() {
        ebiten.SetWindowSize(640, 480)
        ebiten.SetWindowTitle("My Game")
        if err := ebiten.RunGame(&Game{}); err != nil {
            log.Fatal(err)
        }
    }
    ```
  </Step>
</Steps>
